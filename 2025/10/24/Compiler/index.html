Compiling Pipeline

Compiling

  • Lexical Analysis: identify words
  • Syntax Analysis: identify sentences
  • Semantic Analysis: analyze sentences
  • Optimization: editing
  • Code generation: translation

Lexical Analysis

  • Scanning: breaking an input into its smallest meaningful units, called tokens

Token

  • <name,attribute-value>

Syntax Analysis

  • Parser: fit a sequence of tokens into a specified syntax

Grammar

  • defines the valid structure and form of statements (or expressions) in the language
  • Context-Free grammar, CFG

Semantic Analysis

  • use the syntax tree and information in the symbol table to check the source program for semantic consistency with the language definition
    • type checking
    • ……

Optimization

  • Optimization = Analysis + Transformation

must be:

  • Semantics-preserving(safe)
  • Profitable
  • Widely applicable
  • Cheap to perform: time and complexity?

“Optimization” may affect floating-point precision

  • Floating-Point Representation in binary
  • Round-off error
  • Error Propagation

Compiler Structure

Control Flow Graph(CFG)

G = (N, E)

N: nodes, Basic block

E: edges

Basic block(BB)

Definition:

  • A basic block is a maximal sequence of instructions that can be entered only at the first of them and exited only from the last of them
